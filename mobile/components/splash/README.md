# Water Ring Reveal Splash

Expo Go–safe lite splash (~4.5s) on every cold launch: gradient, rotating ring (`WaterRingLite`), car SVG, logo. No Blur, Moti, or animated SVG in the active path.

Legacy full cinematic files (`WaterRingCanvas.jsx`, `DropletField.jsx`) are kept for a future dev build.

## After dependency changes

```bash
npx expo start --clear
```
